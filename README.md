## Quick Start

### Prerequisites

```bash
node >= 18.0.0
npm >= 9.0.0
```

### Installation

```bash
# Clone repository
git clone https://github.com/spookymemes/ai.git

# Navigate to directory
cd spookymemes-ai

# Install dependencies
npm install

# Copy environment template
cp .env.example .env

# Start development server
npm run dev
```
